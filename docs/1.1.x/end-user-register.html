Overview
If you are allowing new trade registrations you will want to create a link to the register.php script so that webmasters can fill out the registration form and submit their information. You can use a standard HTML link pointing to the register.php script. For example, if TradeX is installed in the tx directory of your site, you could use a link like this: <a href="/tx/register.php">Webmasters - Trade Traffic</a> The appearance of the registration form is controlled through the Site Templates, which you can modify to meet your needs.
