Toplists - Manage
Table of Contents
Overview
From this interface you can manage the toplists that you have configured.
Individual Toplist Icons
Each network site configured will have icons at the end of the table row. Those are as follows:
    Click this icon to build the toplist
    Click this icon to edit the toplist. A dialog will appear where you can edit the settings for the toplist.
    Click this icon to delete the toplist
Toolbar Icons
The toolbar at the bottom of this interface has a number of icons you can use to perform different actions. Those are as follows:
    Click this icon to add a new toplist. This will display a dialog where you can enter the settings for the new toplist.
    Click this icon to build several of your toplists at once. Use the checkboxes to select the toplists you want to build.
    Click this icon to delete several of your toplists at once. Use the checkboxes to select the toplists you want to delete.
    Click this icon to access this documentation file
