Link Scanner Results

Introduction

After you have run the link scanner you will be able to display the results to see what actions were taken for the links that were found to be no longer working or breaking your rules. This document will describe how to view the link scanner results and the functions you can use from that interface.

Viewing Link Scanner Results

To view the link scanner results use the Directory > Links > Scan Links function from the control panel menu. This will load a page which will show the current link scanner configurations that are defined. You can click on the icon for any of those configurations to see the results from the last scan that was run.

When you click on one of those icons you will come to a page where you will see the link scanner results and be able to search them. To start your search, enter the term you would like to search for in the text input box and select the field that you would like to search in. Each of the search fields is described below.
    Link ID Thie ID number of the link that was scanned
    Site URL The Site URL of the link that was scanned
    HTTP Status The HTTP status code of the Site URL for the link that was scanned
    Date Scanned The date and time when the link was scanned (YYYY-MM-DD HH:MM:SS format)
    Action Taken The action that was taken on the link. One of Unchanged, Disabled, Deleted, or Blacklisted
You can also choose how the links are sorted by selecting the sorting field from the drop down list. The available sorting fields correspond with the search fields which are described above. Once you have the search values filled in how you want them, press the search button to display the links that meet your criteria.

Link Functions

In the results table you will be able to run a number of different functions on the link scan results that are displayed. Clicking on the icons in the Functions column of the table will allow you to run that function on a single link. You can also check the boxes next to multiple links to perform the functions listed at the bottom of the page. The single link function icons are each described below.
    Click this icon to edit the link. A window will appear where you will be able to edit all of the link settings.
    Click this icon to blacklist the link. A window will appear allowing you to select which items to blacklist.
    Click this icon to re-activate a disabled link
    Click this icon to disable an active link
    Click this icon to send an e-mail message to the link submitter. A window will appear where you can enter the e-mail text
    Click this icon to delete the link. You will be asked to confirm the deletion before it is done.
You can also run a function on multiple links by putting a check in the box next to the links you want, selecting the function to run from the drop down selection field at the bottom of the page, and pressing the Execute button. Each of the available functions are described below.
    E-mail Selected This function will display a window where you can enter your e-mail text and send an e-mail message to the selected links
    Delete Selected This will permanently delete the links you have selected
    Disable Selected This will set the selected links' status to disabled
    Activate Selected This will set the selected links' status to active
    Blacklist Selected This will automatically blacklist the selected links. The site URL, e-mail address, and submit IP will be added to the blacklist
