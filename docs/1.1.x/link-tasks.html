Link Quick Tasks

Introduction

The quick tasks interface allows you to perform search and replace, search and set, and search and delete functions against your link database. This allows you to quickly make mass changes to your links. Each of these functions is described in detail in this document.

Accessing Quick Tasks

The quick tasks functions can be accessed either through the Directory Browsing or Link Search interfaces. In those interfaces you will find the icon in the header bar next to the help icons. You can click on that icon to bring up the quick tasks interface.

Search and Replace

The search and replace function will search for a value in the field you specify and replace it with a new value. For example, if you wanted to update all of your link descriptions so that the word "cars" would become "automobiles" you would use this function. Each of the fields for this function are described below.
    Find Enter the value you want to search for
    In Field Select the field in which the search and replace should be done.
    Descriptions of the fields can be found in the Link Search section of the manual.
    Replace With Enter the replacement value to use
Once you have the fields setup how you want them press the Search and Replace button. The software will update perform this task and then let you know how many replacements were made.

Be careful when using this function. It uses loose matching, so if you enter "tire" for the search value and "radial" for the replacement value, this will match not only the word tire, but also "tiresome", "tired", etc. This would produce replacements of "radialsome" and "radiald", which you most likely do not want.

Search and Set

The search and set function will search for a value in the field you specify and set another field to a specific value. For example, if you wanted to update all of link descriptions that contained the word "truck" and set the keywords to "automobile pickup bedliners offroad" you would use this function. Each of the fields for this function are described below.
    Find Select the field to search in, the search type to perform, and the value to search for.
    Descriptions of the fields can be found in the Link Search section of the manual.
    Set Select the field who's value you want to set, and the value to set that field to.
    Descriptions of the fields can be found in the Link Search section of the manual.
Note that this does not do partial replacements, it sets the field you select to the exact value you enter, overwriting the existing value of that field.
