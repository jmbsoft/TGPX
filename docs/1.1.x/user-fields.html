User Defined Account Fields

Introduction

LinkX allows you to have an unlimited number of user defined fields to store additional information about the user accounts that are created. This document provides an overview of how to enable and use the user defined account fields.

Viewing User Defined Account Fields

To see the currently defined account fields use the Database > Account Fields function from the control panel menu. This will load a page which will show you all of the account fields you have defined thus far. You will see an overview of the field name, label, field type, the options you have set on the field, and functions that can be run.

Adding A User Defined Account Field

To add a new user defined account field, click on the icon in the header bar of the user defined account fields interface. This will display a window where you will be able to enter all of the information for the new field you are adding. Each of those fields are described below.
    Field Name The name to use for the database field. This should contain only English letters, numbers, and the underscore character.
    Label The label that will appear next to this field on the submission forms
    Type The type of HTML input field to use
    Tag Attributes Any custom attribute values you want to appear in the HTML form field tag
    Options A comma separated list of options the user will be presented with (for select fields only)
    Validation The type of validation that should be done on this field
    Validation Error The error message to display if the data provided does not validate
    Display on account creation form Check this box to display this field on the account creation form
    Display on account editing form Check this box to display this field on the account editing form
    Field is required when adding/editing account Check this box if this field is required during account submission/editing
Once you have filled in all of the fields as you want them, press the Add Account Field button. This will add the new account field to the database.

Field Types

There are four different field types that you can select from when adding/updating a user defined link field. Each of those are described below.
    Text A standard one line text input field
    Textarea A multi-line text input field
    Select A drop down selection box
    Checkbox A standard checkbox
Editing an Account Field

Locate the account field you want to delete using the user defined account fields interface. When you locate the account field you want to edit, click on the icon to bring up the account field editing interface. This interface is nearly identical to the account field creation interface, and has the same form fields with the same properties. You can make the changes that you want, and then press the Update Account Field button to save your changes.

Deleting an Account Field

Deleting an account field will also delete all of the data associated with that field for every account in the database

Locate the account field you want to delete using the user defined account fields interface. When you locate the account field you want to delete, click on the icon, which will ask you to confirm the deletion.

You can also delete several account fields at once by putting a check in the box next to the account fields you want to delete and pressing the Delete button at the bottom of the user defined account fields interface.
