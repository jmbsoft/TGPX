Network - Sync Settings
Table of Contents
Overview
From this interface you can sync settings between several TradeX installations. This can be extremely useful for sharing your blacklist, skim schemes, country settings, and more between many TradeX installations.
Settings to Sync
This is where you will select the settings that you want to sync with your other TradeX sites. Simply click on any of the settings to include them in the sync. The settings displayed in blue will be synced, while the settings in grey will not be synced. The available settings that you can sync are:
    Trades - Select this to sync the trade accounts. If the account does not exist on the remote site, it will be created. If it does exist on the remote site, it will be updated with the settings from this site.
    Skim Schemes - Select this to sync your skim schemes. If the skim scheme does not exist on the remote site, it will be created. If it does exist on the remote site, it will be updated with the settings from this site.
    Groups - Select this to sync your groups. The groups from this site will overwrite the groups on the remote site
    Categories - Select this to sync your categories. The categories from this site will overwrite the categories on the remote site
    Blacklist - Select this to sync your blacklist. The blacklist from this site will overwrite the blacklist on the remote site
    Countries - Select this to sync your country qualty settings and weights. This will overwrite the country quality settings and weights on the remote site
    Search Engines - Select this to sync your search engines. The search engines from this site will overwrite the search engines on the remote site
    Trade Rules - Select this to sync your trade rules. The trade rules from this site will overwrite the trade rules on the remote site
    Network Sites - Select this to sync the network sites. If the network site does not exist on the remote site, it will be created. If it does exist on the remote site, it will be updated with the settings from this site.
Trades to Sync
This section will only appear if you have selected the Trades option in Settings to Sync

This is where you will select the trades that you want to sync to your network sites. Simply click on any of the trades to include them in the sync. The trades displayed in blue will be synced, while the trades in grey will not be synced. You can also select trades by category by putting checks in the checkboxes next to the categories of the trades that you want to sync.
Sites to Sync
This is where you will select the sites that you want to sync with. Simply click on any of the sites to include them in the sync. The sites displayed in blue will be synced, while the sites in grey will not be synced. You can also select sites by category or owner by putting checks in the checkboxes next to the categories and/or owners of sites that you want to sync.
Starting the Sync
To start the syncing process, click on the sync icon in the toolbar:
The syncing process can take a while to complete, so be sure that you stay on this page until it has finished. A progress indicator will appear at the top of the page after you click on the sync icon. This will tell you what site is currently being synced and will also show you the sync results. If there were no errors during the sync, this will disappear when it has completed and a confirmation message will display in the top right of your web browser.

If there were errors, you will still be able to see this section after syncing has completed so you can review the results to see which site(s) had problems and what they were.
